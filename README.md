# DNS Query App

DNS query program, retrieves host addresses and email servers.

Comes with a GUI via PyQt5.

## Requirements
1. PyQt5
2. dnslib 0.9.7 (pip install dnslib)

## Usage
Run DNSApp with Python 3.

Simply specify host, and DNS server to run query.
